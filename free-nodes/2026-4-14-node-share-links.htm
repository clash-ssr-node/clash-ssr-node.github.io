一、什么是Shadowrocket？

Shadowrocket是一款专为iOS平台设计的代理客户端，主要用于帮助用户通过多种代理协议（如VMess、Shadowsocks等）实现网络访问的加密与转发。它的核心功能不仅限于绕过网络审查，还包括对流量的加密保护和DNS解析的自定义，保障用户数据的安全和隐私。

作为一款付费软件，Shadowrocket的价格相对合理，但用户可自由选择使用的代理服务器服务，既有免费的，也有收费的服务商，灵活度极高。

二、Shadowrocket的主要功能亮点

  • 多协议支持
    支持VMess、VLESS、Shadowsocks、Socks5等多种主流代理协议，满足不同网络环境下的连接需求。

  • 流量加密保护
    通过内置加密机制，有效防止流量被窃听或篡改，提升通信的安全性。

  • DNS自定义与覆盖
    用户可以自定义DNS服务器，避免DNS污染，提升解析速度和准确性。

  • 规则分流
    支持全球模式和分流模式，允许用户灵活指定哪些流量通过代理，哪些直接连接，节省资源同时提升效率。

  • 智能连接检测
    支持连接测试和自动重连功能，提升连接稳定性。

  • 安装后，务必检查应用权限，尤其是网络权限，确保Shadowrocket能够正常运行。

四、Shadowrocket配置详尽指南

4.1 配置步骤总览

  1. 打开Shadowrocket应用，进入主界面。

  2. 点击“配置”选项，进入配置管理。

  3. 新建或导入代理服务器信息。

  4. 设置规则模式，选择全球代理或分流代理。

  5. 连接并测试代理服务器，确保连接正常。

4.2 添加代理服务器

  • 输入服务器地址和端口号。

  • 选择对应协议（如VMess、Shadowsocks等）。

  • 填写必要的加密和认证信息。

  • 可为服务器设置标签，方便管理。

4.3 配置规则与分流

Shadowrocket允许用户根据需求设定代理规则：

  • 全球模式：所有网络流量均通过代理服务器转发。适合高度受限网络。

  • 分流模式：只有部分指定的域名或IP走代理，其他流量直接访问。提升速度，减少代理压力。

规则列表支持自定义，支持根据应用、IP地址、域名等多维度分类流量。

4.4 连接测试与调试

配置完成后，用户可以通过内置的测试功能检测连接稳定性和速度。若连接失败，应检查服务器信息是否正确，或更换节点进行尝试。

FAQ

Mellow 的 TUN 模式与 SOCKS5 代理区别是什么？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响特定应用。TUN 更适合全局加速、游戏和视频应用，保证所有流量走节点，提高网络稳定性和速度。
Matsuri 节点延迟排序如何优化网络流量？
Matsuri 会定期测速节点延迟，并结合策略组自动选择延迟最低节点。无需手动切换，确保网页、视频和游戏流量通过最优节点，提高访问速度和网络稳定性。
如何使用 Docker 部署 sing-box 或 v2ray？
选择官方或社区维护的 Docker 镜像，编写 docker-compose.yml 指定配置文件挂载、网络模式与端口映射。使用容器管理可以隔离运行环境、便于日志采集并与反向代理（如 Nginx）协同工作。
Mihomo 节点分组如何自动切换？
用户可按延迟、地区或用途将节点分组，并启用策略组。系统会根据节点健康状况和延迟自动切换到最佳节点，保证网页浏览、视频播放和游戏连接稳定顺畅。
VMess 节点无法连接时应如何排查？
首先检查服务器是否在线，其次查看端口、防火墙和 TLS 配置是否正确。还可以通过日志观察 handshake 是否成功，或尝试更换传输协议如 WS、gRPC 来确认问题来源。
如何为 Trojan 配置 ALPN 与 SNI 提升伪装效果？
在服务器与客户端配置中设置合适的 SNI（伪装域名）并启用 ALPN（例如 http/1.1），确保 TLS 握手时使用常见的应用层协议标识，从而增加流量与普通 HTTPS 的相似度，降低被检测概率。
Nekoray 的节点导入是否支持批量 URL？
支持。Nekoray 可通过订阅 URL 批量导入节点，并定期更新。用户无需手动添加每个节点，保证节点列表始终最新，提高使用效率和便利性。
Trojan TLS 伪装域名如何提高抗封锁能力？
通过选择真实可访问域名并使用有效证书，Trojan 流量可以完全模拟 HTTPS 请求。隐藏真实流量特征，降低被封锁和 DPI 检测风险，保证节点长期可用。
如何在 V2Ray 中自定义路由规则？
在 V2Ray 中可以通过修改 config.json 文件中的 routing 部分来自定义路由规则。用户可以根据 IP 段、域名或地理位置来决定流量的去向，从而实现自动分流或访问加速的效果。
VLess 与 Reality 协议可以结合使用吗？
可以。VLess + Reality 是目前最受欢迎的组合之一。它在无证书情况下实现加密传输，同时具备隐蔽性与速度优势，非常适合需要安全访问的高级代理配置。
